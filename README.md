# moesaid.com

Personal portfolio site for Mohamed S Abdrabou — Full Stack Web Developer based in Pittsburgh, PA.

## Tech Stack

- **Build Tool**: [Vite](https://vitejs.dev/) v6
- **CSS Framework**: [TailwindCSS](https://tailwindcss.com/) v4
- **Hosting**: GitHub Pages

## Development

```bash
# Install dependencies
npm install

# Start dev server (http://localhost:5173)
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

The site automatically deploys to GitHub Pages via GitHub Actions when pushing to `main` branch.
